    Informações do Token

    Para concluir a verificação, você precisa adicionar o token abaixo como um registro TXT no serviço de DNS do domínio informado.

    Depois de adicionar o token, clique no botão "Verificar agora" para confirmar a propriedade do domínio.

    Domínio informado
    {{ token.temporary_domain }}
    Token de verificação
    {{ token.token }}
    Expiração
    {{ token.expiration_date|localtime|date:"d \d\e F \d\e Y, H:i" }}
    Como adicionar no DNS

    Adicione o seguinte registro no seu painel de DNS:

    			Tipo: TXT
			Nome: @
			Valor: {{ token.token }}

    Algumas plataformas podem exigir que o nome do registro seja o domínio completo (ex: {{ token.temporary_domain }}) ou apenas "@". Consulte a documentação do seu provedor de DNS.
